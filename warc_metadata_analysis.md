## Методология анализа

Анализ выполнен с помощью утилиты `metawarc` версии 1.1.1 с использованием следующих команд:

1. **analyze** - анализ по MIME-типам и расширениям файлов
2. **metadata** - извлечение метаданных из офисных документов
3. **export** - экспорт HTTP-заголовков

### Использованные команды:
```bash
# Анализ по MIME-типам
python -m metawarc analyze <archive.warc.gz> -m mimes

# Анализ по расширениям
python -m metawarc analyze <archive.warc.gz> -m exts

# Экспорт заголовков
python -m metawarc export <archive.warc.gz> -t headers -o <output.txt>

# Извлечение метаданных
python -m metawarc metadata <archive.warc.gz> -o <output.txt>
```

## Примечания

1. Все архивы содержат только HTML-контент, другие форматы отсутствуют
2. Архив cryptomuseum является самым большим и содержит наибольшее количество записей
2. Архив golden-agetv содержит PHP-страницы с редиректами
3. Экспорт контента (export -t content) не выполнен из-за проблем с кодировкой в некоторых записях
